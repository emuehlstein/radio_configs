# truSDX target

The truSDX has no formal codeplug — bands are configured live via CAT
commands or the front-panel menu. This target renders `ResolvedLayout`
into:

- A **printable crib sheet** (markdown → PDF) with band/mode/freq/tone
  reminders, keyed by group.
- A **CAT command script** (Kenwood-compatible TS-480 subset) for
  operators who want to preset the rig from a serial terminal.

Not yet implemented.
